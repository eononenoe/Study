# 개발 학습 노트

> 회사에서 배운 개념과 기술을 기록하는 공간  
> **코드 없이 순수 학습 내용만 정리합니다**

---

## 이 레포지토리는?

실무에서 부딪힌 문제들과 해결 과정을 **개념 중심**으로 기록합니다.  
회사 기밀 보호를 위해 **실제 코드는 포함하지 않고**, 배운 기술과 개념만 정리합니다.

---

## 기록하는 내용

### 포함하는 것
- 트러블슈팅 (문제 → 원인 → 해결 개념)
- 새로 배운 기술과 개념
- 유용한 검색 키워드
- 참고 자료 링크
- 핵심 학습 포인트

### 절대 포함하지 않는 것
- 실제 프로젝트 코드
- 회사 기밀 정보
- API 키, 서버 정보 등
